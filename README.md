# Tracker

A lightweight personal tracker project for logging activity, reviewing progress, and turning small daily records into useful summaries.

## Project Goal

Tracker is a starter repository for building a simple logging tool. It can be used for habits, study sessions, tasks, workouts, spending, or any other repeated activity where the main workflow is capture first and review later.

## Planned Features

- Add timestamped tracking entries.
- Organize entries by category or tag.
- View summaries by day, week, or month.
- Export records to CSV or Markdown.
- Keep the data model simple enough to inspect and back up manually.

## Suggested Roadmap

1. Decide the first tracking use case.
2. Define the entry format.
3. Add create, list, edit, and delete commands or screens.
4. Add summary views and CSV export.
5. Add tests once the core workflow is stable.
